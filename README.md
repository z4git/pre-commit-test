# pre-commit-test

## Install on Windows

Install Python

`winget install Python.Python.3.8`

Install PIP

`curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py`

`python get-pip.py`

Install pre-commit

`pip install pre-commit`

Enable pre-commit in repo directory

`pre-commit --version`

`pre-commit install`

## Install on Linux

Install pre-commit

`sudo pip install pre-commit`

Enable pre-commit in repo directory

`pre-commit --version`

`pre-commit install`
